[shortcuts](shortcuts) vim



undo: esc+u
close window: :q
    -quits the editor discarding any made changes
close vim: qa!
close vim and save changes: :qw
open links: enter

navigate:
- upwards: j
- downwards: k
- left: h
- right: l

enter insert mode: i
enter appending mode: a


return to normal mode: esc
- deleting characters: x
